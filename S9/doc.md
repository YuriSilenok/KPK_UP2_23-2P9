### Вариант №9. Сервис движения студентов.

Информация требуемая для создания записи о движении студента

| Параметр | Обязательность | Тип | Ограничение | Значение по умолчанию |
|----------|----------------|-----|-------------|-----------------------|
| ID студента | Обязательно | Целое число | строго больше 0 | — |
| Тип движения | Обязательно | Строка | 'expelled', 'reinstated', 'transferred', 'academic_leave', 'back_from_leave' | — |
| Дата начала | Обязательно | Дата | формат ГГГГ-ММ-ДД | — |
| ID группы источника | Не обязательно | Целое число | строго больше 0 | NULL |
| ID группы назначения | Не обязательно | Целое число | строго больше 0 | NULL |
| Основание | Не обязательно | Строка | длина до 255 символов | NULL |
| Номер приказа | Не обязательно | Строка | длина до 50 символов | NULL |
| Дата окончания | Не обязательно | Дата | позже даты начала | NULL |

Комбинация параметров: Тип движения, ID студента и Дата начала должна быть уникальной (у студента не может быть двух одинаковых движений в один день)

**Особые комбинации:**
- Для `transferred` (перевод): обязательны `ID группы источника` и `ID группы назначения`, они не могут быть равны
- Для `academic_leave` (академ. отпуск): `Дата окончания` обязательна и должна быть позже `Даты начала`
- Для `reinstated` (восстановление) или `back_from_leave` (выход из отпуска): обязателен `ID группы назначения`
- Для `expelled` (отчисление): обязателен `ID группы источника`

Входные параметры (для изменения движения)

| Параметр | Обязательность | Тип | Ограничение | Значение по умолчанию |
|----------|----------------|-----|-------------|-----------------------|
| Тип движения | Не обязательно | Строка | как при создании | — |
| Дата начала | Не обязательно | Дата | формат ГГГГ-ММ-ДД | — |
| ID группы источника | Не обязательно | Целое число | строго больше 0 | NULL |
| ID группы назначения | Не обязательно | Целое число | строго больше 0 | NULL |
| Основание | Не обязательно | Строка | длина до 255 символов | NULL |
| Номер приказа | Не обязательно | Строка | длина до 50 символов | NULL |
| Дата окончания | Не обязательно | Дата | позже даты начала | NULL |

Выходные данные (при создании/получении движения)

| Параметр | Тип |
|----------|-----|
| ID записи | Целое число |
| ID студента | Целое число |
| ФИО студента | Строка |
| Тип движения | Строка |
| Дата начала | Дата |
| ID группы источника | Целое число или NULL |
| Название группы источника | Строка или NULL |
| ID группы назначения | Целое число или NULL |
| Название группы назначения | Строка или NULL |
| Основание | Строка или NULL |
| Номер приказа | Строка или NULL |
| Дата окончания | Дата или NULL |

Вернет истину (True) если запись о движении была удалена, иначе ложно (False)

Параметры запроса (для получения списка движений)

| Параметр | Тип | Описание |
|----------|-----|----------|
| ID студента | Целое число | Фильтр по конкретному студенту |
| Тип движения | Строка | Фильтр по типу ('expelled', 'transferred' и т.д.) |
| ID группы источника | Целое число | Фильтр по исходной группе |
| ID группы назначения | Целое число | Фильтр по целевой группе |

| Дата начала по | Дата | Движения не позднее указанной даты |
| Лимит | Целое число | Максимум 100, по умолчанию 20 |
| Смещение | Целое число | Для пагинации, по умолчанию 0 |

Результат содержит только активные записи (не удалённые)

#### Правила валидации

1. Нельзя создать новое движение, если у студента есть незакрытый академический отпуск (без `Даты окончания` или с `Датой окончания` в будущем)
2. Нельзя отчислить студента, который находится в академическом отпуске
3. При переводе ID группы источника и ID группы назначения не могут совпадать
