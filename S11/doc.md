### Функции сервиса: Сервис дисциплин
#### 1. Добавление группы.

| Параметр | Обязательность | Тип | Ограничение | Значение по умолчанию |
|---|---|---|---|---|
| Код дисциплины | Обязательно |Строка | Длина от 3 до 20 символов | — |
| Название дисциплины | Обязательно |Строка | Длина от 1 до 200 символов | — |
| Краткое название | Не обязательно | Строка | Длина от 0 до 30 символов | Пустая строка |
2. Получение списка активных дисциплин по параметрам:
   - По коду дисциплины,
   - По названию дисциплины,
   - По краткому названию дисциплины.
3. Получение информации о дисциплине.
   - С помощью этой функции можно получить следующую информацию о дисциплине: Код дисциплины, название дисциплины, краткое название дисциплины.
4. Удаление дисциплины
   - Подрозумевает под собой изменение статуса активности группы на неактивную
5. Изменение информации о определенной дисциплине
   - Изменение краткого названия дисциплины.
