### Вариант №3. Role Service (Сервис ролей).
#### Добавить роль.

Информация требуемая для создания роли
| Параметр | Пояснение | Обязательность | Тип | Ограничение | Значение по умолчанию |
|---|---|---|---|---|---|
|role name|Название роли (Админ, Директор, Завуч, Преподаватель, Студент, Родитель)|Обязательно|Строка| | |
|access level|Уровень доступа роли (чем выше уровень роли, тем больше прав)|Обязательно|Целое число|больше 0||
|role description|Описание роли (доступный функционал, пример для роли Студент: |Не обязательно|Строка||`NULL`|
