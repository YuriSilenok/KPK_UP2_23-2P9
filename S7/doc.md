### Вариант №7. Сервис групп.
#### Добавить группу.

Информация требуемая для создания группы
| Параметр | Обязательность | Тип | Ограничение | Значение по умолчанию |
|---|---|---|---|---|
| Год создания группы | Обязательно | Целое число | от 2000 |  |
| Номер группы | Обязательно | Целое число | от 1 |  |
| Префикс | Обязательно | Строка |  |  |
| Шифр направления подготовки | Обязательно | Строка | вида `NN.NN.NN` (цифры, разделённые точками) |  |
| После какого класса | Обязательно | Литерал (9, 11) | 9 или 11 |  |
| id куратора | Не обязательно | Целое число | больше 0 | `NULL` |

Комбинация, параметров: `Год создания группы`, `Номер группы`, `Префикс` и `После какого класса`, должна быть уникальной

#### Измененить группу по ID

Входные параметры
| Параметр | Обязательность | Тип | Ограничение | Значение по умолчанию |
|---|---|---|---|---|
| id куратора | Обязательно | Целое число | больше 0 | `NULL` |

Выходные данные
| Параметр | Тип |
|---|---|
| Год создания группы | Целое число | 
| Номер группы | Целое число |
| Префикс | Строка |
| Шифр направления подготовки  | Строка |
| После какого класса | Литерал (9, 11) | 
| id куратора | Целое число или `NULL` | 
| Наименование группы | Строка (пример `3-2П9`) | 
| Количество студетов | Целое число |
| Список студетов | Список целых чисел |

#### Удаление группы по ID

Вернет истинну (`True`) если группа была закрыта, иначе ложно (`False`)

#### Информация о группах

Параметры запроса
| Параметр | Тип | Описание |
|---|---|---|
| Номер курса (перечисление) | Целое число | "1,2,3" ; "1,3" |
| Номер курса (минимальное значение) | Целое число | от "1" |
| Номер курса (максимальное значение) | Целое число | до "1" |
| Количество студентов (перечисление) | Целое число | "1,2,3" ; "1,3" |
| Количество студентов (минимальное значение) | Целое число | от "1" |
| Количество студентов (максимальное значение) | Целое число |  до "1" |
| Префикс | Строка | 
| Шифр направления подготовки | Строка | вида `NN.NN.NN` (цифры, разделённые точками) |
| После какого класса | Литерал (9, 11) | 9 или 11 |
| id куратора | Целое число | 0 - если куратор не назначен |

Результат содержит только активные группы
