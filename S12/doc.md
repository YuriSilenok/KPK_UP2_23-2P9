### Функции сервиса: Сервис учебного плана
#### 1. Добавление плана.

| Параметр | Обязательность | Тип | Ограничение | Значение по умолчанию |
|---|---|---|---|---|
| ID дисциплины | Обязательно | Целое число | больше 0 | `NULL` |
| Номер семестр | Обязательно | Целое число | от 1 до 2 | — |
| Теория часы | Обязательно | Целое число |  от 0 до 180 | — |
| Практика часы | Обязательно | Целое число |  от 0 до 180 | — |
| СР часы | Обязательно | Целое число |  от 0 до 40 | — |
| Форма отчётности | Обязательно | Строка | `экзамен` или `зачёт` | — |

2. Получение списка активных планов по параметрам:
   - По ID дисциплины,
   - По номеру семестра,
   - По форме отчётности.
3. Получение информации о дисциплине.
   - С помощью этой функции можно получить следующую информацию о дисциплине: Код дисциплины, название дисциплины, краткое название дисциплины.
4. Удаление дисциплины
   - Подрозумевает под собой изменение статуса активности группы на неактивную
5. Изменение информации о определенной дисциплине
   - Изменение краткого названия дисциплины.
