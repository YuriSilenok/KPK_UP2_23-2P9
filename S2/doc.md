### Вариант №2. Сервис профилей.
#### Добавить профиль.

Информация требуемая для создания профиля:
| Параметр | Пояснение | Обязательность | Тип | Ограничение | Значение по умолчанию |
|---|---|---|---|---|---|
| id_profil | ID профиля | Обязательно | Целое число |  |  |
| first_name | Фамилия | Обязательно | Строка |  |  |
| second_name | Имя | Обязательно | Строка |  |  |
| last_name | Отчество | Не обязательно | Строка |  | 'NULL' |
| telephone | Номер телефона | Обязательно | Цело число | = 11 |  |
| e-mail | Электронная почта | Обязательно | Строка |  |  |
| photo | Фотография | Обязательно | Строка |  |  |

Параметр'ID профиля' должен быть уникальным.

При удачном добавлении профиля вернуть информацию (в виде таблицы с столбцами):
| Параметр | Тип |
|---|---|
| id_profil | Целое число |
| first_name | Строка |
| second_name | Строка |
| last_name | Строка |
| telephone | Цело число |
| e-mail | Строка |
| photo | Строка |

#### Изменить профиль по ID.

Входные параметры:
| Параметр | Пояснение | Обязательность | Тип | Ограничение | Значение по умолчанию |
|---|---|---|---|---|---|
| id_profil | ID профиля | Обязательно | Целое число |  |  |
| first_name | Фамилия | Не обязательно | Строка |  | 'NULL' |
| second_name | Имя | Не обязательно | Строка |  | 'NULL' |
| last_name | Отчество | Не обязательно | Строка |  | 'NULL' |
| telephone | Номер телефона | Не обязательно | Цело число | = 11 | 'NULL' |
| e-mail | Электронная почта | Не обязательно | Строка |  | 'NULL' |
| photo | Фотография | Не обязательно | Строка |  | 'NULL' |

Выходные параметры:
| Параметр | Тип |
|---|---|
| id_profil | Целое число |
| first_name | Строка |
| second_name | Строка |
| last_name | Строка |
| telephone | Цело число |
| e-mail | Строка |
| photo | Строка |

#### Удаление группы по ID.
Вернет истинну ('True') если профиль был закрыт, иначе вернет ложность ('False')

#### Получить информацию о профиле по ID.
Информация возвращаемая пользователю в случае удачного поиска профиля по ID:
| Параметр | Тип |
|---|---|
| id_profil | Целое число |
| first_name | Строка |
| second_name | Строка |
| last_name | Строка |
| telephone | Цело число |
| e-mail | Строка |
| photo | Строка |
| profil_act | Буллевое значение |

#### Получить список профилей по параметрам.
Параметры запроса:
| Параметр | Пояснение | Тип | Описание |
|---|---|---|---|
| first_name | Фамилия | Строка | Получение списка профилей с значением определенной фамилии |
| second_name | Имя | Строка | Получение списка профилей с значением определенного имени |
| last_name | Отчество | Строка | Получение списка профилей с значением определенного отчества |
| telephone | Номер телефона | Цело число | Получение списка профилей с значением определенного телефона |
| e-mail | Электронная почта | Строка | Получение списка профилей с значением определенной электронной почты |
| profil_act | Активность профиля | Буллевое значение | "True" - активные профили; "False" - неактивные профили |

При удачном запросе вернуть пользователю список профилей с информацией:
| Параметр | Тип |
|---|---|
| id_profil | Целое число |
| first_name | Строка |
| second_name | Строка |
| last_name | Строка |
| telephone | Цело число |
| e-mail | Строка |
| photo | Строка |

#### Настройки уведомлений.
Информация требуемая для настройки уведомлений:
| Параметр | Пояснение | Обязательность | Тип | Ограничение | Значение по умолчанию |
|---|---|---|---|---|---|
|id_profil | ID профиля | Обязательно | Целое число |  |  |
|notification_turn | Включение уведомлений | Обязательно | Буллевое значение |  | 'True' |

Информация возвращаемая пользователю в случае удачной настройки уведомлений:
| Параметр | Тип |
|---|---|
|id_profil | ID профиля | Целое число |
|notification_turn | Включение уведомлений | Буллевое значение |
