### Вариант №2. Сервис профилей.
#### Добавить профиль.

Информация требуемая для создания профиля (в виде таблицы с столбцами):
| Параметр | Обязательность | Тип | Ограничение | Значение по умолчанию |
|---|---|---|---|---|
| ID профиля | Обязательно | Целое число |  |  |
| Фамилия | Обязательно | Строка |  |  |
| Имя | Обязательно | Строка |  |  |
| Отчество | Не обязательно | Строка |  |  |
| Номер телефона | Обязательно | Цело число | = 11 |  |
| Электронная почта | Обязательно | Строка |  |  |
| Фотография | Обязательно | 'PNG' |  |  |

Комбинация параметров: 'ID роли' и 'ID профиля' должна быть уникальной.

При удачном добавлении профиля вернуть информацию (в виде таблицы с столбцами):
| Параметр | Тип |
|---|---|
| ID профиля | Целое число |
| Фамилия | Строка |
| Имя | Строка |
| Отчество | Строка |
| Номер телефона | Цело число |
| Электронная почта | Строка |
| Фотография | 'PNG' |

#### Изменить профиль по ID.

Входные параметры:
| Параметр | Обязательность | Тип | Ограничение | Значение по умолчанию |
|---|---|---|---|---|
| ID профиля | Обязательно | Целое число |  |  |
| Фамилия | Не обязательно | Строка |  | 'NULL' |
| Имя | Не обязательно | Строка |  | 'NULL' |
| Отчество | Не обязательно | Строка |  | 'NULL' |
| Номер телефона | Не обязательно | Цело число | = 11 | 'NULL' |
| Электронная почта | Не обязательно | Строка |  | 'NULL' |
| Фотография | Не обязательно | 'PNG' |  | 'NULL' |

Выходные параметры:
| Параметр | Тип |
|---|---|
| ID профиля | Целое число |
| Фамилия | Строка |
| Имя | Строка |
| Отчество | Строка |
| Номер телефона | Цело число |
| Электронная почта | Строка |
| Фотография | PNG |

#### Удаление группы по ID.
Вернет истинну ('True') если профиль был закрыт, иначе вернет ложность ('False')

#### Получить информацию о профиле по ID.
Информация возвращаемая (в виде таблицы с столбцами) в случае удачного поиска профиля по ID:
| Параметр | Тип |
|---|---|
| ID профиля | Целое число |
| Фамилия | Строка |
| Имя | Строка |
| Отчество | Строка |
| Номер телефона | Цело число |
| Электронная почта | Строка |
| Фотография | 'PNG' |
| Активность профиля | Буллевое значение |

#### Получить список профилей по параметрам.
Параметры запроса:
| Параметр | Тип | Описание |
|---|---|---|
| Фамилия | Строка | Получение списка профилей с значением определенной фамилии |
| Имя | Строка | Получение списка профилей с значением определенного имени |
| Отчество | Строка | Получение списка профилей с значением определенного отчества |
| Номер телефона | Цело число | Получение списка профилей с значением определенного телефона |
| Электронная почта | Строка | Получение списка профилей с значением определенной электронной почты |
| Активность профиля | Буллевое значение | "True" - активные профили; "False" - неактивные профили |

При удачном запросе вернуть список профилей (в виде таблицы с столбцами):
| Параметр | Тип |
|---|---|
| ID профиля | Целое число |
| Фамилия | Строка |
| Имя | Строка |
| Отчество | Строка |
| Номер телефона | Цело число |
| Электронная почта | Строка |
| Фотография | 'PNG' |

Информация требуемая для настройки уведомлений:
| Параметр | Обязательность | Тип | Ограничение | Значение по умолчанию |
|---|---|---|---|---|

Информация возвращаемая (в виде таблицы с столбцами) в случае удачной настройки уведомлений:
| Параметр | Тип |
|---|---|
| ID профиля | Целое число |
| Включение уведомлений | Буллевое значение |
