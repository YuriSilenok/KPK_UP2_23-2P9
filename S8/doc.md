### Вариант №8. Сервис подгрупп.
#### Добавить подгруппу.

Информация требуемая для создания подгруппы
| Параметр | Обязательность | Тип | Ограничение | Значение по умолчанию |
|---|---|---|---|---|
| id группы | Обязательно | Целое число | больше 0 |  |
| Номер подгруппы | Обязательно | Целое число | от 1 |  |
| Список студентов | Не обязательно | Список целых чисел |  | Пустой список |

#### Измененить подгруппу по ID

Входные параметры
| Параметр | Обязательность | Тип | Ограничение | Значение по умолчанию |
|---|---|---|---|---|
| Список студентов | Обязательно | Список целых чисел |  |  |

Выходные данные
| Параметр | Тип |
|---|---|
| id группы | Целое число |  
| Наименование подгруппы | Строка (пример `3-2П9-1`) | 
| Количество студетов | Целое число |
| Список студетов | Список целых чисел |

#### Удаление подгруппы по ID

Вернет истинну (`True`) если подгруппа была закрыта, иначе ложно (`False`)

#### Информация о подгруппе по ID

Информация о группе для пользователя
| Параметр | Тип |
|---|---|
| id подгруппы | Целое число | 
| id группы | Целое число |
| Номер подгруппы | Целое число |
| Наименование подгруппы | Строка (пример `3-2П9-1`) |
| Количество студетов | Целое число |
| Список студетов | Список целых чисел |

#### Информация о подгруппах

Параметры запроса
| Параметр | Тип | Описание |
|---|---|---|
| ID группы | Целое число | Равно указанному номеру |
| Номер подгруппы | Целое число | Равно указанному номеру |
| Наименование подгруппы | Строка | Поиск по частичному совпадению без учета регистра |
| Количество студентов | Строка | "1" - равно 1, "1,3" - от 1 до 3, "1," - от 1, ",3" - до 3. Включительно |

Результат содержит только активные подгруппы

Информация о подгруппах для пользователя
| Параметр | Тип |
|---|---|
| id подгруппы | Целое число |
| id группы | Целое число |
| Год создания подгруппы | Целое число |
